Сведения о ставках ОСМС

В регистре сведений Сведения о ставках ОСМС указаны ставки взносов и отчислений на обязательное медицинское страхование в разрезе видов доходов.

Ставки могут быть внесены в целом для всех организаций информационной базы, в таком случае графа «Налогоплательщик» не заполняется (Данный реквизит доступен для редактирования с января 2020 года). Если размер ставки различается для разных организаций, то следует вносить ставки по каждой организации в отдельности. Приоритет определения ставок, используемых для исчисления сумм взносов и отчислений на ОСМС, следующий:

  • если найдена строка с заполненной графой «Налогоплательщик», то используется значение ставки из этой строки (в соответсвии с указанным видом дохода);
  • иначе используется значение ставки из строки регистра сведений, в которой графа «Налогоплательщик» не заполнена (в соответсвии с указанным видом дохода).
