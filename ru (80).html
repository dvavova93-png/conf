Удержания организаций

План видов расчета «Удержания организаций»

Содержит перечень так называемых "базовых" расчета, т.е. таких начислений, от суммы которых считается текущее удержание. Используется для начислений со способом расчета процентом.

Реквизиты:
  • Наименование и Код - задаются пользователем самостоятельно в зависимости от используемого в организации подхода к кодированию удержаний, на расчет результатов не влияют.
  • Способ расчета - определяет формулу, которая вычисляется при автоматическом расчете удержания, обязателен к заполнению. Ниже приведена таблица доступных способов расчета.

Закладка Расчеты:

(доступна, если выбран способ расчета "Процентом" или «Исполнительный лист процентом»)

Закладка Учет:
  • Способ отражения в бухучете - содержит правило отражения результатов начисления в бухгалтерском учете.
  • Объединять в расчетном листке - если флажок установлен, то при формировании расчетных листков все удержания такого вида будет "свернуты" в одну строку. Рекомендуется указывать, если в течение одного месяца по одному и тому же сотруднику производится несколько раз одинаковое удержание (например, ежедневно удержание за питание по фактической стоимости обеда), в таком случае установка признака позволит сократить объем выводимой информации в расчетном листе.

Закладка Шкала ставок

(доступна, если выбран способ расчета "Почтовый сбор")

Закладка Прочее:

Содержит список видов удержаний, используемых при расчете заработной платы сотрудников организаций. Список может дополняться пользователем по мере необходимости самостоятельно, кроме того, в нем уже содержатся наиболее часто используемые виды удержаний, такие как: "Удержание по исполнительному листу (процентом и суммой)", "Почтовый сбор по исполнительным листам", которые являются предопределенными и не могут быть удалены из списка удержаний. При создании новых видов удержаний для упрощения заполнения его свойств рекомендуется вводить новые виды путем копирования предопределенных.

 Для разного рода процентных удержаний. Если на закладке «Расчеты» установлены флажки «Вычитать из базы ИПН» и/или «Вычитать из базы ОПВ», то расчетная база уменьшается на сумму ИПН/ОПВ.
Почтовый сбор По шкале в зависимости от суммы почтового перевода. Для удержания оплаты за почтовый перевод алиментов. Шкала задается на закладке Шкала ставок.
  • Вычитать из базы ИПН - если флажок установлен, то при расчете суммы удержания из расчетной базы (закладка Прочее) будет исключена сумма исчисленного индивидуального подоходного налога за текущий месяц, определяемая по регистру накопления "ИПН расчеты с бюджетом".
  • Вычитать из базы ОПВ - если флажок установлен, то при расчете суммы удержания из расчетной базы (закладка Прочее) будет исключена сумма исчисленных обязательных пенсионных взносов за текущий месяц, определяемая по регистру накопления "ОПВ расчеты с фондами".
  • Вычитать из базы ВОСМС - если флажок установлен, то при расчете суммы удержания из расчетной базы (закладка Прочее) будет исключена сумма исчисленных взносов на обязательное социальное медицинское страхование за текущий месяц, определяемая по регистру накопления "ВОСМС расчеты с фондами"

Содержит шкалу ставок почтового сбора в зависимости от размера почтового перевода (суммы удержания по исполнительному листу).
