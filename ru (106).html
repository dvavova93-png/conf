Регистр накопления «ОПВР расчеты с фондами»

Регистр предназначен для учета взаиморасчетов по обязательным профессиональным пенсионным взносам.

Измерения регистра:

  • Дата - дата регистрации взаиморасчетов;
  • Физ. лицо - физическое лицо, к которому относится запись регистра;
  • Структурная единиц - обособленное подразделение или организация, определяющая структурную единицу, по которой будут отражены взаиморасчеты в налоговых отчетах (форма 200.00 и т.д.);
  • Месяц налогового периода - месяц, за который исчислялись ОППВ;
  • Вид платежа - вид платежа по ОППВ (непосредственно взносы, пени, штрафы и т.д.);
  • Организация - организация, в которой осуществляются расчеты по ОППВ;
  • Налогоплательщик - организация - налоговый агент, предоставляющая налоговую отчетность по ОППВ;
  • Взнос - сумма ОППВ;
  • Вид строки - характеристика записи регистра, определяющая в результате какой операции (исчисления, перечисления, возврата) введена текущая строка регистра;
  • Дата начала и Дата окончания - используются при расчете пени за несвоевременное перечисление взносов, определяют период, за который рассчитывается пени.


Приход по регистру осуществляется при исчислении и возврате ошибочно перечисленный сумм ОППВ. Расход по регистру производится при перечислении взносов в фонды.

Для анализа взаиморасчетов по пенсионным взносам предназначен отчет «Структура задолженности организаций по ОППВ».
