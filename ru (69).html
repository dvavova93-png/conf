Регистр сведений "Размеры отчислений в резервы"

Регистр предназначен для задания размеров отчислений в резервы по вознаграждениям сотрудникам.

Измерения регистра:

  • Организация - организация, в которой происходит формирование резерва;
  • Резерв - вид формируемого резерва - элемент справочника "Резервы".


Ресурсы регистра:

  • % отчислений в резерв - процент отчислений в резерв по вознаграждениям сотрудникам (на оплату отпуска, премии по итогам работы за год и т.п.) от суммы базовых видов начислений, указанных в форме справочника "Резервы";
  • % СН - процент отчислений в резерв для покрытия затрат по социальному налогу, связанному с начислением вознаграждения сотрудникам;
  • % СО - процент отчислений в резерв для покрытия затрат по социальном отчислениям, связанным с начислением вознаграждения сотрудникам;
  • % ОППВ - процент отчислений в резерв для покрытия затрат по обязательным профессиональным пенсионным взносам, связанным с начислением вознаграждения сотрудникам;
  • % ООСМС - процент отчислений в резерв для покрытия затрат по взносам на обязательное социальное медицинское страхование;
  • % ОПВР - процент отчислений в резерв для покрытия затрат по обязательным пенсионным взносам работодателя;
  • Счет учета - счет учета резерва в бухгалтерском учете.

Если при формировании резерва не требуется учитывать социальный налог, социальные отчисления и/или обязательные профессиональные пенсионные взносы, то соответствующие ресурсы следует оставить пустыми. Для определения размера отчислений в резерв можно проанализировать результаты расчетов за предыдущий период и указать фактический процент затрат.

Настройки из регистра "Размеры отчислений в резервы" используются при автозаполнении документа "Отражение зарплаты в регл. учете". Сумма формируемого резерва определяется по следующим формулам:

Сумма резерва по вознаграждениям = Сумма базовых видов расчета * %отчислений;
Сумма резерва СН = Сумма резерва по вознаграждениям * %СН;
Сумма резерва СО = Сумма резерва по вознаграждениям * %СО;
Сумма резерва ОППВ = Сумма резерва по вознаграждениям * %ОППВ;
Сумма резерва ООСМС = Сумма резерва по вознаграждениям * %ООСМС;

Сумма резерва ОПВР = Сумма резерва по вознаграждениям * %ОПВР.


Общая сумма резерва = Сумма резерва по вознаграждениям + Сумма резерва СН + Сумма резерва СО + Сумма резерва ОППВ + Сумма резерва ООСМС + Сумма резерва ОПВР.

У видов начислений, которые выплачиваются за счет сформированных резервов, необходимо в форме настройки указать способ отражения в бухгалтерском учете, в дебете которого указан счет учета резерва и значение субконто "Резервы" содержит соответствующий вид резерва.

Для анализа результатов формирования резерва по вознаграждениям сотрудникам предназначен отчет "Справка-расчет "Резервы по вознаграждениям сотрудникам".
