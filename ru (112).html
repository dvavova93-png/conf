Регистр накопления "НДС к возмещению"


Регистр предназначен для учета суммы НДС, относимого в зачет.

Данные записи регистра:

  • Налогоплательщик - организация, либо подразделение организации, в случае, если оно является самостоятельным плательщиком НДС.
  • Контрагент - поставщик ТМЗ, работ, услуг.
  • ТМЗ - наименование приобретенного товара, работы, услуги.
  • Вид поступления - вид операции приобретения в Декларации по НДС.
  • Вид налога НДС - тип налога на добавленную стоимость - НДС, НДС по импорту, НДС с измененным сроком уплаты, НДС методом зачета.
  • Вид оборота - содержит информацию о том, для целей какого оборота (облагаемого, освобожденного либо общего) приобретен товар (работа, услуга).
  • Ставка НДС - ставка налога на добавленную стоимость.
  • Сумма без налога - сумма поступления ТМЗ без учета налогов.
  • Уплаченный НДС - НДС, уплаченный при приобретении ТМЗ (указанный в счет-фактуре поставщика).
  • Сумма НДС - сумма НДС, относимая в зачет. Отличается от суммы уплаченного НДС в случаях, когда в соответствии с законодательством НДС не может быть отнесен в зачет, например при приобретении товаров для личного потребления и т.д.


На основании данных регистра заполняются Декларация по НДС (форма 300.00) и приложение №8 формы 300.00.
