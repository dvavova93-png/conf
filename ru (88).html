(Не использовать) Свойства объектов

Планы видов характеристик "Свойства объектов"

Справочник вызывается для ввода и редактирования информации из пункта «Главное меню» / «Все функции» / «Планы видов характеристик» / «Свойства объектов».

Для свойства объектов могут быть назначены различные типы значения свойств.

На этапе конфигурирования в список свойств объектов могут быть добавлены предопределенные значения свойств, категорий или характеристик.

В процессе работы с типовой конфигурацией пользователь может добавлять свойства, имеющие различные типы значений.

Добавление нового вида свойства производится в отдельном диалоговом окне, которое открывается при нажатии на клавишу INS или при выборе соответствующей пиктограммы на панели управления.

Реквизиты:
  • Наименование - задается пользователем самостоятельно в зависимости от используемого в организации подхода.
  • Код – заполняется автоматически при записи элемента справочника, а также может быть откорректирован вручную.

Закладка Назначения свойств:

Можно указать конкретные объекты или группы объектов, для которых будет предназначено данное свойство.

Например: свойство "Коммуникабельность" целесообразно назначить для группы работников "Менеджеры".

В список объектов, для которых предназначено свойство, можно добавить несколько групп справочника или даже указать конкретных физических лиц, если назначаемое свойство имеет смысл только для них. Если в список объектов, для которых предназначено свойство, не выбран ни один элемент, то данное свойство назначается всем объектам указанного справочника или документа.

Закладка Значения свойств:

Вводится список возможных значений для данного вида свойства.
