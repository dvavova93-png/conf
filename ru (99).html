Регистр накопления "СО расчеты с фондами"


Регистр предназначен для учета взаиморасчетов по обязательным социальным отчислениям. Данные регистра детализируют информацию на счете 3210 "Обязательства по пенсионному обеспечению" бухгалтерского учета.

Измерения регистра:

  • Дата - дата регистрации взаиморасчетов;
  • Месяц налогового периода - месяц, за который исчислялись социальные отчисления;
  • Физ. лицо - физическое лицо, к которому относится запись регистра;
  • Структурная единица - обособленное подразделение или организация, определяющая структурную единицу, по которой будут отражены взаиморасчеты в налоговых отчетах (форма 200.00 и т.д.);
  • Вид платежа - вид платежа по социальным отчислениям (непосредственно отчисления, пени, штрафы и т.д.);
  • Налогоплательщик - организация - налоговый агент, предоставляющая налоговую отчетность по социальным отчислениям.


Ресурсы регистра:

  • Отчисление - сумма социальных отчислений.


Реквизиты регистра:

  • Вид строки - характеристика записи регистра, определяющая в результате какой операции (исчисления, перечисления, возврата) введена текущая строка регистра;
  • Дата начала и Дата окончания - используются при расчете пени за несвоевременное перечисление социальных отчислений, определяют период, за который рассчитывается пени.


Приход по регистру осуществляется при исчислении и возврате ошибочно перечисленный сумм социальных отчислений. Расход по регистру производится при перечислении социальных отчислений в фонд социального страхования.

Для анализа взаиморасчетов по социальным отчислениям предназначен отчет "Структура задолженности организаций по СО".
