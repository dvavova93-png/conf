Регистр сведений "Способы отражения расходов по амортизации нематериальных активов (бухгалтерский учет)"


В регистре хранятся данные о соответствии способов отражения расходов по амортизации нематериальных активов организаций для бухгалтерского учета.

Данные из регистра используются при расчете амортизации.

Регистр является периодическим, данные которого изменяются следующими документами:

  • "Изменение способов отражения расходов по амортизации НМА";
  • "Принятие к учету НМА".
