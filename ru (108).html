Регистр накопления "ОПВ подлежит перечислению в фонды"


Регистр предназначен для учета обязательных пенсионных взносов, подлежащих перечислению в пенсионные фонды. Используется для отражения в регламентированной налоговой отчетности суммы ОПВ, подлежащей перечислению, а также для расчета пени за несвоевременное перечисление ОПВ.

Измерения регистра:

  • Дата - определяет дату регистрации взаиморасчетов;
  • Физ. лицо - физическое лицо, к которому относится запись регистра;
  • Структурная единица - обособленное подразделение или организация, определяющая структурную единицу, по которой будут отражены взаиморасчеты в налоговых отчетах (форма 200.00 и т.д.);
  • Месяц налогового периода - месяц, за который исчислялись ОПВ;
  • Месяц выплаты доходов - месяц, в котором были выплачены доходы, с которых исчислены ОПВ. Используется для определения сроков перечисления при расчете пени за несвоевременное перечисление ОПВ;
  • Налогоплательщик - организация, предоставляющая налоговую отчетность по обязательным пенсионным взносам.


Ресурсы регистра:

  • Взнос - сумма ОПВ.


Реквизиты регистра:

  • Вид строки - характеристика записи регистра, определяющая в результате какой операции (удержания, перечисления, возврата) введена текущая строка регистра.


Приход по регистру осуществляется при удержании (отнесении ОПВ к суммам подлежащим перечислению), а также возврате ошибочно перечисленных взносов. Расход по регистру производится при перечислении обязательных пенсионных взносов.
