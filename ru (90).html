Настройки пользователей

Для каждого пользователя могут быть заданы персональные настройки и значения по умолчанию.

В качестве настроек и значений по умолчанию для конкретного пользователя могут быть заданы настройки, которые в дальнейшей работе будут автоматически проставляться в документах, что поможет более оперативно обрабатывать документы и рационально использовать рабочее время. Значения по умолчанию задаются в группе "Основные значения для подстановки в документы и справочники":

  • Основная единица по классификатору, Основная ставка НДС– единица измерения и ставка НДС, которые будут проставляться по умолчанию при оформлении новых записей позиций номенклатуры.
  • Основная валюта взаиморасчетов. Валюта взаиморасчетов, которая будет использована в качестве валюты по умолчанию при оформлении новых договоров с контрагентами.
  • Основная организация, основной склад, основная касса, основное подразделение, основной покупатель, основной поставщик, основной тип цен продажи. Значения, которые будут использованы при оформлении новых документов.
  • Основной ответственный. Сотрудник, который контролирует действия пользователя и отвечает за правильность оформления документов. В качестве ответственного сотрудника может быть, например, менеджер по закупкам, который отвечает за оформление операции покупки. В то же время конкретные документы могут выписывать различные пользователи: документы поступления выписывает кладовщик, а документы оплаты регистрирует кассир или бухгалтер. Информация об ответственном сотруднике, заданным по умолчанию для пользователя, в дальнейшем будет заполняться при оформлении новых документов. В качестве ответственного сотрудника может быть назначен и сам пользователь.
  • Вывод сумм прописью при печати в документах. При установке данного флага при печати документа "Расходный кассовый ордер" будет заполняться строка с суммой прописью.
  • Запрашивать вид операции при вводе нового. При вводе новых документов, у которых присутствует реквизит "Вид операции", будет происходить запрос конкретного вида операции.


В группе "Другие настройки" определяются настройки пользователя при открытии документов и формировании отчетов:

  • При открытии документов позиционироваться на дату. У пользователя при открытии документов курсор будет автоматически позиционироваться на дату документа.
  • Формировать отчеты без данных по валютам. При формировании отчетов данные будут формироваться в валюте регламентированного учета, без учета данных по валютам.


В группе "Панель функций" устанавливаются настройки видимости панели функций и текущей страници при открытии:

  • Показывать панель функций.
  • Текущая страница панели функций.


Дополнительно могут быть введены следующие настройки:

  • Запрашивать подтверждение при закрытии. При закрытии конфигурации будет выведено сообщение, которое нужно подтвердить.
  • Контроль корреспонденции счетов. При установке данного флага при проведении документов будет контролироваться правильность корреспонденции счетов бухгалтерского и налогового учета в документа.
  • Показывать в документах счета учета. Показывать движения при проведении. Определяют режим работы документов при проведении.
  • Основная дата начала отчетов. Дата, которая будет проставляться в отчеты в качестве начальной даты интервала, за которую будет построен отчет. Если конечная дата в настройках отчета не указана, то отчет будет строится по текущую дату.
  • Основной каталог файлов пользователя. Имя каталога, в котором будут храниться внешние файлы. Этот каталог будет использоваться по умолчанию при работе пользователя с внешними файлами.
  • Печать документов без предварительного просмотра. При установке флага документы будут выводиться сразу на принтер, без предварительного просмотра. Данная настройка действует только для тех форм, которые установлены в документе, как формы по умолчанию.
  • Показывать "Быстрое освоение" при начале работы системы. Производит запуск соответствующей обработки при запуске программы.
  • Учет по всем организациям. При установке флага, программа позволяет вести учет по всем организациям, которые были созданы в справочнике "Организации".
