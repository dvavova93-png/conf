Регистр сведений «История обмена данными»

Регистр сведений «История обмена данными» предназначен для хранения истории обмена данными между узлами РИБ.

Данный регистр имеет одно измерение «Настройка обмена», предназначенное для хранения ссылки на справочник «Настройки обмена данными».

Также данный регистр содержит следующие ресурсы:

  • Дата загрузки - дата последней загрузки данных
  • Дата выгрузки - дата последней выгрузки данных
  • Результат последней загрузки - содержит информацию о том, завершилась ли загрузка корректно.
  • Результат последней выгрузки - содержит информацию о том, завершилась ли выгрузка корректно.
  • Отложенные движения - содержит информацию по результату отложенных движений
  • Фоновая загрузка - наличие данных фоновой загрузки
  • Дата начала последней загрузки - дата начала последней загрузки данных
  • Дата начала последней выгрузки - дата начала последней выгрузки данных
