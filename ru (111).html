Регистр накопления "Взаиморасчеты с бюджетом по НДС с измененным сроком уплаты"


Регистр предназначен для учета взаиморасчетов с бюджетом по налогу на добавленную стоимость, срок уплаты которого при импорте товаров был изменен.

Данные записи регистра:

  • Налогоплательщик - организация, либо подразделение организации, в случае, если оно является самостоятельным плательщиком НДС.
  • Вид поступления - вид операции поступления в декларации по НДС.
  • Вид налога НДС - тип налога на добавленную стоимость - НДС, НДС по импорту, НДС с измененным сроком уплаты, НДС методом зачета.
  • Контрагент - поставщик ТМЗ, работ, услуг.
  • ГТД - указывается номер грузовой таможенной декларации.
  • Срок платежа - срок (измененный), установленный для погашения налога.
  • Сумма НДС - сумма налога на добавленную стоимость.


На основании данных регистра заполняется приложение 300.03 к Декларации по НДС.
