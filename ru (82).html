Журнал проводок (бухгалтерский учет)

Регистр бухгалтерии «Журнал проводок (бухгалтерский учет)»

Регистр сведений вызывается для ввода и редактирования информации из пункта «Главное меню» / «Все функции» / «Регистры бухгалтерии» / «Журнал проводок (бухгалтерский учет)».

Журнал проводок является основной формой списка регистра бухгалтерии «Бухгалтерский учет» и представляет собой удобный инструмент для просмотра, отбора, поиска и редактирования проводок.

В Журнале проводок фиксируются проводки, выполненные с помощью документов. В нем каждая операция отображается одной строчкой, содержащей наиболее важную информацию (дату, номер, содержание, сумму). Журнал проводок позволяет просмотреть существующие проводки операций в виде общего списка.

Для поиска нужной информации целесообразно использовать настройки отбора и сортировки списка.

Реквизиты отбора «Счет» и «Организация» вынесены в форму журнала проводок исключительно с целью упрощения этих отборов и не имеют никакого преимущества перед аналогичными реквизитами стандартного диалога отбора и сортировки этой формы. Более того, они однозначно связаны с ними.

Поле Счет

Если указать номер счета, то можно отсортировать проводки по выбранному счету.

Поле Организация

При выборе организации можно отобрать проводки по выбранной организации.

Кнопка Перейти к документу

Открывает документ, на основании которого выполнена проводка; также по двойному щелчку мыши на проводке возможен переход к документу, сформировавшему эту проводку.

Регистр использует план счетов «Типовой план счетов бухгалтерского учета» и поддерживает корреспонденции. Это означает, что каждая запись регистра имеет дебетовую и кредитовую часть.

Измерения
  • Организация - балансовое измерение регистра обеспечивает ведение учета в разрезе юридических лиц.
  • Валюта - измерение регистра является небалансовым и обеспечивает учет по валютным счетам в разрезе отдельных наименований иностранных валют.

Ресурсы
  • Сумма - балансовый ресурс является основным и обеспечивает учет в национальной валюте на всех счетах плана счетов; это относится и к предопределенным счетам и к тем, которые введены в режиме 1С:Предприятие; по отдельным видам субконто на счетах суммовой учет может не вестись.
  • Валютная сумма - ресурс является небалансовым, а значит, может вестись выборочно, для отдельных счетов; такие счета имеют признак «Валютный»; ведение валютного учета по каждому виду субконто на таких счетах настраивается отдельно; ведение валютного учета по счету в целом и по отдельным видам субконто дает возможность получать остатки и обороты в валюте в разрезе валют.
  • Количество - ресурс является небалансовым и ведется только для тех счетов, для которых имеет смысл учет в натуральных измерителях; такие счета имеют признак «Количественный»; ведение количественного учета по каждому виду субконто на таких счетах настраивается отдельно; количественный учет по счету и субконто ведется без разделения по единицам измерения; предполагается, что для каждого вида актива это учет в одной основной единице измерения.

Реквизиты
  • Номер журнала - номер журнала является вспомогательным реквизитом для условного разделения проводок по разделам учета (товары, основные средства, операции по банку и кассе и т.п.); может использоваться для быстрого отбора проводок (по нему установлено индексирование).
  • Содержание - содержание представляет собой краткое описание сути проводки.
