Регистр сведений "Сведения о группах фиксированных активов"

В регистре содержатся сведения о параметрах налогового учета фиксированных активов - предельной норме амортизации, применяемой норме амортизации, проценте для вычета расходов на ремонт основных средств.

Сведения должны быть указаны для каждой группы фиксированных активов, по которым ведется налоговый учет в организации. Для активов первой группы учет ведется пообъектно, в связи с этим, в случае, если для объектов первой группы параметры налогового учета отличаются от тех, что установлены для группы в целом необходимо задать их для каждого объекта первой группы.
