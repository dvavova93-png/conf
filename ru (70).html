Размеры вычетов ИПН

Регистр сведений «Размеры вычетов ИПН

Регистр сведений вызывается для ввода и редактирования информации из пункта «Главное меню» / «Все функции» / «Регистры сведений» / «Размеры вычетов ИПН».

Регистр сведений предназначен для хранения сведений по размерам вычетов ИПН, начиная с 2019 года. (До 2019 года автоматизированные виды вычетов хранятся в справочнике «Виды вчетов ИПН» в виде предопределенных элементов).

В регистре сведений присутствуют следующие элементы:

  • Стандартный 14 МРП (1 МЗП)
  • Стандартный вычет (лицам с инвалидностью, участникам ВОВ)
  • Стандартный вычет (родителям лиц с инвалидностью, приемным родителям)
  • Вычет для многодетной семьи (для каждого родителя)
  • Вычет для многодетной семьи (для одного из родителей)

Для нового элемента регистра обязательным для заполнения является поле «Вид вычета ИПН». Если указанное поле не заполнено, система не позволит записать элемент.

Реализована возможность добавления, изменения и удаления нового элемента. Для нового вида размера вычета ИПН создается карточка со следующими полями:период предоставления, вид вычета ИПН, вид предела вычета, размер (предел вычета).
