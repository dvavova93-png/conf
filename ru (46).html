Регистр сведений "Способы отражения расходов по амортизации основных средств (бухгалтерский учет)"


В регистре хранятся данные о соответствии способов отражения расходов по амортизации основных средств организаций для бухгалтерского учета.

Данные из регистра используются при расчете амортизации.

Регистр является периодическим, данные которого изменяются следующими документами:

  • "Ввод начальных остатков ОС";
  • "Изменение способов отражения расходов по амортизации ОС";
  • "Принятие к учету ОС".
