Товары с пониженной ставкой пошлин (Перечень изъятий)

Регистр сведений используется для хранения списка товаров, у которых код товарной номенклатуры внешнеэкономической деятельности (ТНВЭД) включен в Перечень изъятий - перечень товаров, ввозимых на территорию Республики Казахстан из третьих стран, не являющихся государствами-членами Евразийского экономического союза (ЕАЭС), к которым применяются пониженные ставки пошлин.

Для перехода к просмотру утвержденного Перечня изъятий используется команда Перечень изъятий.

Регистр может быть заполнен элементами справочника Номенклатура, для которых указан код ТНВЭД, входящий в Перечень изъятий.

Механизм заполнения регистра сведений (подменю Заполнить):

  • Заполнить товарами из перечня изъятия - все ранее заполненные данные очищаются и регистр заполняется всеми товарными позициями, у которых заполнен код ТНВЭД, входящий в Перечень изъятий;
  • Добавить товарами из перечня изъятий - заполненные данные не очищаются, а дополняются недостающими товарными позициями, у которых заполнен код ТНВЭД, входящий в Перечень изъятий.
