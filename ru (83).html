Журнал проводок (налоговый учет)

Регистр бухгалтерии «Журнал проводок (налоговый учет)»

Регистр сведений вызывается для ввода и редактирования информации из пункта «Главное меню» / «Все функции» / «Регистры бухгалтерии» / «Журнал проводок (налоговый учет)».

Регистр бухгалтерии «Налоговый учет» используется для ведения налогового учета по корпоративному подоходному налогу.

Регистр использует план счетов «План счетов налогового учета» и поддерживает корреспонденции. Это означает, что каждая запись регистра имеет дебетовую и кредитовую часть. Несмотря на то, что все счета налогового учета забалансовые, корреспонденции счетов являются наглядным способом отражения хозяйственной операции, хотя и не обязательны.

Журнал проводок является основной формой списка регистра бухгалтерии «Налоговый учет» и представляет собой удобный инструмент для просмотра, отбора, поиска и редактирования проводок.

В Журнале проводок фиксируются проводки, выполненные с помощью документов. В нем каждая операция отображается одной строчкой, содержащей наиболее важную информацию (дату, номер, содержание, сумму). Журнал проводок позволяет просмотреть существующие проводки операций в виде общего списка.

Для поиска нужной информации целесообразно использовать настройки отбора и сортировки списка.

Реквизиты отбора «Счет» и «Организация» вынесены в форму журнала проводок исключительно с целью упрощения этих отборов и не имеют никакого преимущества перед аналогичными реквизитами стандартного диалога отбора и сортировки этой формы. Более того, они однозначно связаны с ними.

Поле Счет

Если указать номер счета, то можно отсортировать проводки по выбранному счету.

Поле Организация

При выборе организации можно отобрать проводки по выбранной организации.

Кнопка Перейти к документу

Открывает документ, на основании которого выполнена проводка; также по двойному щелчку мыши на проводке возможен переход к документу, сформировавшему эту проводку; также по двойному щелчку мыши на проводке возможен переход к документу, сформировавшему эту проводку.

Измерения

Ресурсы

Измерения
  • Организация - балансовое измерение регистра обеспечивает ведение учета в разрезе юридических лиц - налоговых агентов.
  • Вид учета - измерение регистра является небалансовым и обеспечивает учет в целях СБУ 11 «Учет по подоходному налогу». Вид учета может принимать одно из трех значений:
    • НУ - оценка активов, доходов и расходов по данным налогового учета;
    • ПР - постоянные разницы в оценка активов, доходов и расходов;
    • ВР - временные разницы в оценка активов, доходов и расходов.

Ресурсы
  • Сумма - балансовый ресурс является основным и обеспечивает учет в национальной валюте на всех счетах плана счетов. Это относится и к предопределенным счетам и к тем, которые введены в режиме 1С:Предприятие; по отдельным видам субконто на счетах суммовой учет может не вестись.
  • Количество - ресурс является небалансовым и ведется только для тех счетов, для которых имеет смысл учет в натуральных измерителях. Такие счета имеют признак «Количественный». Ведение количественного учета по каждому виду субконто на таких счетах настраивается отдельно; количественный учет по счету и субконто ведется без разделения по единицам измерения; предполагается, что для каждого вида актива это учет в одной основной единице измерения.

Реквизиты

  • Номер журнала - является вспомогательным реквизитом для условного разделения проводок по разделам учета (товары, основные средства, операции по банку и кассе и т.п.); может использоваться для быстрого отбора проводок (по нему установлено индексирование).
  • Содержание - содержание представляет собой краткое описание сути проводки.
