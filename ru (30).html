Регистр сведений «Объекты доступа документов»

Данный регистр сведений предназначен для ограничения доступа к документам в журналах документов. Записи в регистр осуществляются при записи документов.

Измерения регистра:

  • Документ ссылка - документ, для которого определяется объект доступа;
  • Вид объекта доступа - вид объекта доступа, по которому сработает ограничение: Организации, Склады, Физические лица, Кассы;
  • Объект доступа - непосредственный объект доступа из справочника «Физические лица», «Организации», «Кассы», «Склады».
