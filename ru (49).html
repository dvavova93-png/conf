Регистр сведений «Способ оценки запасов»

Регистр предназначен для установки способа оценки запасов:

- ФИФО - метод, при котором себестоимость определяется по способу "первый пришел первый ушел";

- ЛИФО - метод, при котором себестоимость определяется по способу "последний пришел первый ушел";

- По средней за месяц - метод, при котором себестоимость товара рассчитывается на момент выбытия, а в конце месяца корректируется с учетом всех поступлений товара;

- По средней на момент выбытия - метод, при котором себестоимость товара рассчитывается на момент выбытия и в конце месяца не корректируется.

Для каждого счета учета ТМЗ может быть установлен свой способ оценки в разрезе «Организации» и периода . В случае, если для счета не задано значение способа оценки, считается, что для данного счета установлен способ «По средней на момент выбытия». Если установлен способ оценки «ЛИФО» или «ФИФО» для счета добавляется аналитика «Партия».

Способ оценки для счета может быть изменен, только в том случае, если в периоде действия данной настройки еще не было оборотов по указанному счету.
